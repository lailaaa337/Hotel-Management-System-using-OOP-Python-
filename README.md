# Hotel Management System (OOP - Python)

## Overview

This project is a simple hotel management system built using Python and object-oriented programming. It simulates how a hotel handles guests, rooms, staff, and bookings through a menu-based interface.

## What I did

* Designed multiple classes like Hotel, Guest, Room, Staff, and Booking
* Applied OOP concepts such as inheritance, encapsulation, and polymorphism
* Built functions to manage rooms (add, search, delete, sort)
* Added booking functionality with check-in and check-out dates
* Implemented staff salary calculation

## How it works

The program runs in the terminal and shows a menu where the user can choose different actions like adding guests, managing rooms, or booking a room. Each option triggers specific functionality inside the system.

## Notes

This project was mainly for practicing OOP concepts and understanding how real-life systems can be modeled using classes and objects.
